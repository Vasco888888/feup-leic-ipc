# MyTradingPal

This repository contains the deliverables produced for **Interação Pessoa-Computador** (Human-Computer Interaction) at FEUP, for the **MyTradingPal** project — an AI-driven investing platform that pairs real-money and simulated trading with AI characters that explain their decisions in plain language, aimed at young, mobile-first, beginner investors.

The project was developed across three phases, covering user/task analysis, low/mid-fi prototyping and heuristic evaluation, and hi-fi prototyping with user evaluation.

## Repository Structure

- `Phase 1/`: User and Task Analysis — project idea, questionnaire highlights, PACT analysis, report and presentation.
- `Phase 2/`: Low/mid-fidelity prototype and heuristic evaluation — report and presentation.
- `Phase 3/`: Hi-fidelity prototype and user evaluation — report, presentation and supporting task/SUS score images.
